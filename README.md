# Book Store SQL Analysis Project

## 📌 Description
This project is based on a book store database using SQL.  
It includes queries to analyze sales, customers, and inventory.

## 🛠️ Tables Used
- Books
- Customers
- Orders

## 🔍 Key Queries
- Total revenue calculation
- Most expensive book
- Customer spending analysis
- Remaining stock after orders

## 🚀 Skills Used
- SQL (JOIN, GROUP BY, HAVING)
- Data Analysis
